Survival Server Mods

Check out the Mods for the Friend Survival Server


These steps assume that you've been granted permission from Survival Server Admins to join the server. If you haven't, please ask an admin before completing these steps.

Installing the Mods for the Survival Server:

  1. Download and Run the Forge 1.12.2-14.23.5.2854 Launcher (If that launcher doesn't work download a different one from the forge website.)
  2. Download the Server's Mods and unzip the folder.
  3. Place all the files in the folder in the mods folder in your .minecraft folder. Note: If you don't see a mods folder, launch Forge 1.12.2 and try again.
  4. Join the server on Forge 1.12.2 (The IP will be DM'd to you by Survival Server Admins.)

If you need help installing the mods, make a Support Ticket with the ?new command.
